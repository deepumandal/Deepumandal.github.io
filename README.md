## Run Locally

Clone the project

```bash
  git clone https://github.com/Deepumandal/new-portfolio.git
```

Go to the project directory

```bash
  cd my-project
```

Install dependencies

```bash 
  npm install || yarn 
```

Start the server

```bash
  npm run start || yarn run start
```
